** TaskFlow Pro**

A modern, responsive task management web application designed to simplify daily task planning and progress tracking.

 **Overview**

TaskFlow Pro helps users:

Add and manage daily tasks

Track completion percentage using a dynamic circular progress indicator

Monitor task priority (Low, Medium, High)

Automatically assign system date & time when tasks are created

Visualize priority distribution with interactive charts

Export tasks as CSV files

The application is fully client-side and stores data using LocalStorage, meaning tasks persist even after refreshing the page.

✨ Features
📋 Task Management

Add tasks with title, priority, and completion %

Automatic system date & time assignment

Delete tasks instantly

Drag-and-drop task reordering (SortableJS)

📊 Progress Tracking

Circular progress indicator (like battery %)

Global average progress bar

Real-time statistics dashboard:

Total Tasks

Completed Tasks

Pending Tasks

📈 Analytics

Transparent background charts for dark UI consistency
